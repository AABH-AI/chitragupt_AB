# Sprint 0 — Discovery & Documentation

**Goal:** Capture all foundational knowledge before writing a line of code. Sprint 0 produces documents, not deployables.

By the end of Sprint 0 the team has clear answers to four questions: what are we building, who is it for, what rules govern the system, and what technology choices are non-negotiable going forward.
